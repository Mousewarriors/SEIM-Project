### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/siem-soc-training.git
cd siem-soc-training/soc-app

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### AI Assistant Setup (Optional)

```bash

ollama pull qwen2.5:14b

# Ollama runs automatically on localhost:11434
# The app will detect it and enable the AI assistant
```

### OpenSearch Integration (Optional)

```bash
# Create the index
```
